# IMAGES NEEDED FOR LANDING PAGE

Place these images in the `assets/images/` folder:

## 1. webdura-logo.png
- **Size:** 200px width × 60px height (approximately)
- **Format:** PNG with transparent background
- **What:** Your Webdura company logo
- **Where to get:** From your existing website or marketing materials
- **Used:** In header navigation

## 2. dashboard-preview.png
- **Size:** 1200px width × 800px height (or similar ratio)
- **Format:** PNG or JPG
- **What:** Screenshot of your Creative Performance Analyzer dashboard
- **How to create:**
  1. Open your analyzer dashboard in browser
  2. Make sure it shows sample data (top metrics, creative cards, etc.)
  3. Take full-page screenshot
  4. Crop to show just the dashboard (remove browser chrome)
  5. Save as PNG
- **Used:** In hero section as product preview

## 3. favicon.png
- **Size:** 32×32px or 64×64px
- **Format:** PNG
- **What:** Small icon that appears in browser tab
- **How to create:**
  1. Take your Webdura logo
  2. Resize to 32×32px or 64×64px
  3. Save as PNG
- **Used:** Browser tab icon

## Quick Tips:

### If You Don't Have Images Ready:

**The page will still work!** It shows placeholder text where images would be.

**For logo:** The text "Webdura Tools" appears if logo is missing
**For dashboard:** A gray placeholder box appears
**For favicon:** Browser shows default icon

### Recommended Tools for Creating Images:

- **Screenshots:** 
  - Mac: Cmd + Shift + 4
  - Windows: Windows Key + Shift + S
  - Chrome: DevTools (F12) → Mobile device emulator
  
- **Image Editing:**
  - Free: GIMP, Paint.NET, Photopea (browser-based)
  - Paid: Photoshop, Sketch, Figma
  
- **Quick favicon:**
  - Use: https://www.favicon-generator.org/
  - Upload your logo, download 32x32px version
